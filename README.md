# Real-time Catalog
 
Kyan Dillan Verado - 223400001

Yosafat Bagas Herlianka - 233402003

Real-time Catalog adalah aplikasi web real-time yang memungkinkan manajemen dan monitoring harga barang secara live. Sistem ini dibangun dengan Go di backend dan WebSocket untuk komunikasi real-time antara server dan client.

Cara menjalankan aplikasi ini:
1. Copy link github ini, dan gunakan visual studio kemudian buka terminal baru
2. Di terminal ketik git clone https://github.com/KyanDV/Real-time-Marketplace.git
3. Kemudian ketik Go mod tidy
4. Dan ketik Go run main.go
5. Terdapat 2 halaman, admin dan viewer. Buka halaman viewer terlebih dahulu kemudian halaman admin.
