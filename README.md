Real-time Catalog adalah aplikasi web real-time yang memungkinkan manajemen dan monitoring harga barang secara live. Sistem ini dibangun dengan Go di backend dan WebSocket untuk komunikasi real-time antara server dan client.

Cara menjalankan aplikasi ini:

Copy link github ini, dan gunakan visual studio kemudian buka terminal baru
Di terminal ketik git clone https://github.com/KyanDV/Real-time-Marketplace.git
Kemudian ketik Go mod tidy
Dan ketik Go run main.go
Terdapat 2 halaman, admin dan viewer. Buka halaman viewer terlebih dahulu kemudian halaman admin.
